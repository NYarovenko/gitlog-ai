## КРОК 1 — Встанови Node.js (5 хв)

1. Зайди на https://nodejs.org
2. Натисни кнопку "LTS" — завантажиться установник
3. Запусти установник, натискай "Next" до кінця
4. Відкрий термінал:
   - Windows: Win+R → cmd → Enter
   - Mac: Cmd+Space → Terminal → Enter
5. Перевір встановлення:
      node --version
   Має показати: v18.x.x або вище

## КРОК 4 — Налаштуй файл з ключами (3 хв)

1. Відкрий папку gitlog-ai
2. Знайди .env.local.example
3. Скопіюй і перейменуй в .env.local
4. Відкрий у Блокноті (Windows) або TextEdit (Mac)
5. Заповни:

   ANTHROPIC_API_KEY=sk-ant-[ТВІ_КЛЮЧ]

   NEXT_PUBLIC_GUMROAD_STARTER_URL=https://НІКНЕЙМ.gumroad.com/l/STARTER

## КРОК 5 — Деплой на Vercel (10 хв)

### 5.1 Завантаж на GitHub
1. Зайди на https://github.com → зареєструйся
2. "+" → "New repository" → назви gitlog-ai → Create
3. Відкрий термінал у папці проекту:

   Windows:
   cd C:\Users\ТВО_ІМ_Я\Downloads\gitlog-ai

   Mac:
   cd ~/Downloads/gitlog-ai

   Потім по черзі:
   git init
   git add .
   git commit -m "first commit"
   git branch -M main
   git remote add origin https://github.com/ЛОГІН/gitlog-ai.git
   git push -u origin main

### 5.2 Деплой
1. Зайди на https://vercel.com → Sign up with GitHub
2. "New Project" → обери gitlog-ai → Import
3. Розгорни "Environment Variables"
4. Додай всі змінні з .env.local по одній
5. "Deploy" → чекай 2-3 хвилини
6. Скопіюй URL (наприклад gitlog-ai.vercel.app)

### 5.3 Онови APP_URL
1. Vercel → Settings → Environment Variables
2. NEXT_PUBLIC_APP_URL → https://gitlog-ai.vercel.app
3. Deployments → Redeploy
